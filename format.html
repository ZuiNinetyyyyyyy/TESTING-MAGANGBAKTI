Format File JSON

Buat file .json dengan struktur berikut, lalu upload di halaman utama.

📌 Struktur Dasar

[
  {
    "id": 1,
    "question": "Apa ibu kota Indonesia?",
    "options": [
      "Surabaya",
      "Jakarta",
      "Bandung",
      "Yogyakarta"
    ],
    "answer": 1
  },
  {
    "id": 2,
    "question": "15 × 8 = ?",
    "options": ["100", "110", "120", "130"],
    "answer": 2,
    "explanation": "15 × 8 = 120. Kamu bisa hitung: 15 × 8 = 10×8 + 5×8 = 80 + 40 = 120."
  }
]

📋 Penjelasan Field

Field Tipe Keterangan
id number Opsional. ID unik soal.
question string Wajib. Teks pertanyaan.
options array Wajib. Daftar pilihan jawaban (min 2, maks 6).
answer number Wajib. Index jawaban benar di array options (mulai dari 0).
explanation string Opsional. Penjelasan jawaban (tampil di halaman pembahasan).

💡 Tips

• answer: 0 berarti pilihan A (index pertama).

• Kamu bisa membungkus array dalam objek: {"questions": [...]} — keduanya diterima.

• Soal bisa berjumlah berapapun, tidak ada batasan.

• Aktifkan fitur Acak urutan soal di halaman utama agar soal diacak setiap kuis dimulai.
